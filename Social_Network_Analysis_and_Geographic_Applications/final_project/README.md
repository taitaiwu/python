# 連結地理系的橋樑：探討團體活動參與對地理系學生人際網路影響與傳承結構

> 社會網路分析與地理應用 期末專題分析報告  
> 地理四 S1143022 蔡家晨 / 地理四 S1143042 吳侑謙 / 地理三 S1243038 張庭瑜 / 資工三 S1254059 吳佳泰

---

## 一、研究背景摘要

本研究採用隸屬網路（Affiliation Network）的 **Two-mode to One-mode** 分析框架，蒐集地理系大二至大四共 **45 位**學生的團體活動參與紀錄，涵蓋 8 項系上活動，並輔以熟識度問卷，從 **Event（活動）** 與 **Actor（學生）** 雙重視角，探討以下核心問題：

1. 哪些活動具備橋樑（Bridge）性質，吸引不同群體的學生共同參與？
2. 哪些學生是網路中的關鍵節點，維持了系上社群的整體穩定性？
3. 共同參與活動是否真的促進了學生之間的熟識？

---

### 2.3 Event 分析結論

**高重合活動群（Jaccard > 0.35）——Bridge Events**

| 活動對 | Jaccard | 解讀 |
|--------|---------|------|
| 系學會 & 迎新 | 0.59 | 成員高度重疊，系學會成員幾乎都經歷迎新 |
| 迎新 & 地理營 | 0.46 | 迎新為地理營的主要人才來源 |
| 系學會 & 地理營 | 0.41 | 系學會與地理營共享穩定的幹部群 |
| 迎新 & 系羽 | 0.42 | 系羽成員多由迎新活動帶入 |

**結構洞（Structural Hole）——男排 × 女排 (Jaccard = 0.00)**

男排與女排的 Jaccard 係數為 **0.00**，表示兩者**完全沒有共同成員**，各自形成封閉的運動小圈，是系上活動網路中唯一的完整結構洞。

**核心橋樑活動：迎新**

迎新在所有活動中與其他活動的平均 Jaccard 係數最高（排除自身為 0.26），且同時與系學會（0.59）、地理營（0.46）、系羽（0.42）保持高重合，驗證了迎新作為系上**跨群體接觸平台**的核心橋樑地位。

**活動叢集分析**

根據 Jaccard 矩陣可辨識出兩個活動叢集：
- **叢集 A（社交型）**：系學會、迎新、地理營、探索營、系羽——成員高度重疊，形成系上社交核心
- **叢集 B（運動型）**：男排、女排、系環——相對獨立，與社交型活動重合度低

## 三、Actor 視角分析

### 3.1 雙層網路比對（熟識問卷 vs. Two-mode 活動網路）

本研究將熟識度問卷（Layer 2，友誼網路）與活動共同參與矩陣（Layer 1，隸屬網路）對照，分為四種情形：

| 值 | 意義 | 詮釋 |
|----|------|------|
| **1** | 有共同參與 **且** 互相熟識 | 活動成功促進熟識 |
| **2** | 無共同參與，但互相熟識 | 透過其他管道建立關係 |
| **3** | 有共同參與，但**不**互相熟識 | 活動接觸未轉化為熟識 |
| **4** | 無共同參與，且不熟識 | 完全無交集 |

從對照矩陣的整體樣態觀察：
- **值 1（活動+熟識）** 佔多數，說明共同參與活動**確實是促進熟識的主要管道**。
- **值 3（活動但不熟識）** 亦普遍存在，特別在大型活動（系學會、迎新）中，人數多導致部分參與者雖共場卻未深入交流。
- **值 2（無活動但熟識）** 極少，顯示課堂以外的私下建交較罕見。
- **值 4（完全無交集）** 主要出現在**蔡宗佑**一列，反映其在整體網路中的孤立狀態，與其中心性分析結果一致。

> **結論**：共同參與活動是系上人際網路形成的**必要但非充分條件**——活動規模越大，人際轉化效率越低；小型、互動密集型活動（如探索營、地理營）的熟識轉化率相對較高。

## 四、雙層網路整合分析

### 4.1 活動參與 → 熟識形成的轉換效率

| 情形 | 代碼 | 數量分布 | 解讀 |
|------|------|----------|------|
| 活動共參與 + 熟識確認 | 1 | 最多 | 活動有效促進熟識 |
| 無共同活動 + 熟識確認 | 2 | 極少 | 僅少數透過非活動管道建立關係 |
| 活動共參與 + 不熟識 | 3 | 次多 | 大型活動中的「假性接觸」 |
| 無共同活動 + 不熟識 | 4 | 少數 | 蔡宗佑列中較多，反映其孤立 |

### 4.2 活動驗證熟識假設的結論

本研究原假設「**共同參與活動 = 熟識**」並非完全成立：

- **支持假設（值1）**：大多數情況下，共同參與確實帶來熟識，特別在小型活動（探索營、地理營）中效果最顯著。
- **反例（值3）**：系學會（93人）與迎新（83人）中存在大量「有共同參與但不熟識」的情形，說明**活動規模越大，熟識轉化率越低**。
- **例外橋樑（值2）**：少數熟識建立於非活動場合，暗示某些學生的人際網絡形成管道更為多元。

## 五、總結與發現

### 5.1 Event 層面

| 結論 | 依據 |
|------|------|
| **迎新是最具橋樑性質的活動** | 與系學會（0.59）、地理營（0.46）、系羽（0.42）均有高 Jaccard 係數 |
| **男排與女排存在完全結構洞** | Jaccard = 0.00，兩者完全無共同成員 |
| **活動形成兩大叢集** | 社交型（系學會/迎新/地理營）vs. 運動型（男排/女排/系環） |
| **系學會規模最大但熟識效率有限** | 93人參與但成員間不一定相互熟識 |

### 5.2 Actor 層面

| 結論 | 依據 |
|------|------|
| **張庭瑜是系上最核心的橋樑 Actor** | 四項中心性指標全部第一 |
| **75.6% 的學生屬於高凝聚核心群體** | K-core = 21，34 節點進入最大核心 |
| **吳侑謙具「弱連帶橋樑」特性** | 介數高（排名 9）、度中心性低（排名 43） |
| **蔡宗佑是網路中最孤立的節點** | 四項中心性均墊底，未進入 K-core |
| **多重活動參與者更易成為網路核心** | K-core 成員多為積極參與多種活動者 |

### 5.3 對研究假設的回應

> **「共同參與活動是否促進熟識？」**

**答：部分成立。** 活動共參與是熟識的重要前提，但並非充分條件。活動的**規模、互動密度、持續時間**決定了熟識轉化的效率。小型、長時間共處的活動（如地理營、探索營）比大型儀式性活動（系學會、迎新）更有效地促成深度熟識。

> **「誰是維持地理系整體穩定的重要節點？」**

**答：張庭瑜、薛智升、李翊豪、羅亦軒**構成系上社會網路的四大核心支柱，一旦其中任何人離開系上（如畢業），將對網路凝聚度產生顯著影響，具有重要的傳承意義。
