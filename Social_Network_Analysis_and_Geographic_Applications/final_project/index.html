Research Overview

研究概覽

從 Event（活動）與 Actor（學生）雙重視角，結合隸屬網路與熟識度問卷，解析系上人際網路的橋樑結構。

EVENT 活動視角

  • 蒐集大二至大四學生參與 8 項系上活動的 0/1 矩陣
  • Two-mode → One-mode 轉換，建立活動間鄰近矩陣
  • 以 Jaccard 係數正規化成員重合度
  • 辨識具橋樑性質的活動與活動叢集

ACTOR 學生視角

  • 透過熟識度問卷建立有向的友誼網路（DiGraph）
  • 計算四項中心性：Degree / Closeness / Betweenness / Eigenvector
  • 以 K-Core 分析找出核心群體
  • 雙層網路比對：活動共參與 vs. 實際熟識

BRIDGE 核心問題

  • 哪個 Event 具備橋樑性質，連結不同群體？
  • 哪個 Actor 扮演跨群體的關鍵橋樑角色？
  • 共同參與活動是否真的促進了熟識？
  • 誰是維持系上社群穩定的重要節點？

DATA 資料來源

  • train_data.csv：學生 × 活動參與矩陣（93 筆）
  • familiarity_questionnaire.csv：45 × 45 熟識度問卷
  • 由系學會長提供活動資料，輔以問卷調查補充
  • 熟識定義：共處同空間會閒聊、見到會開口打招呼

🔗
活動形成兩大叢集——社交型叢集（系學會、迎新、地理營、探索營、系羽）與運動型叢集（男排、女排、系環），前者成員高度重疊。
📊
系學會是最大活動但連結效率相對稀薄——93 人參與使得人均互動密度低；小型活動如地理營（Jaccard 0.29 對探索營）反而促成更紮實的熟識。

Findings & Conclusions

研究總結

整合 Event 與 Actor 雙重視角的分析結果，回應研究發問。

核心橋樑 Event

迎新是最具橋樑性質的活動（平均 Jaccard 最高），同時連結系學會、地理營、系羽三大活動群，是跨群體接觸的關鍵平台。

完整結構洞

男排 × 女排的 Jaccard 係數為 0.00，是系上活動網路唯一的完整結構洞。如能有人同時參與兩者，將發揮極強的橋樑效果。

核心橋樑 Actor

張庭瑜在所有四項中心性指標中均位列第一，是系上社會網路無可爭議的核心支柱，對系上傳承延續具關鍵影響。

弱連帶橋樑

吳侑謙是典型的 Weak Tie Bridge——雖然直接認識的人不多（度中心性排 43），但介數中心性排名第 9，橫跨兩個不同群體之間。

活動 → 熟識的效率

共同參與活動是熟識的必要但非充分條件。小型高互動活動（地理營、探索營）熟識轉化率遠高於大型儀式性活動。

核心群體凝聚度

75.6% 的學生屬於 21-Core 緊密核心，顯示系上社群整體凝聚度相當高。建議後續關注未進入核心的 11 位邊緣學生的融入狀況。

對未來的啟示

本研究建議系學會在規劃活動時，特別關注小型互動型活動（如探索營、地理營）的舉辦頻率——相較於大型活動，這類活動更能有效促進深度熟識，進而強化系上社會網路的凝聚力與傳承延續性。在橋樑 Actor 畢業後，如何維持既有的跨群體連結，是系上值得思考的傳承挑戰。
