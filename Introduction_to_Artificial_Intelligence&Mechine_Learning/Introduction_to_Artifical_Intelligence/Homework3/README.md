# Homework3: 期中報告

> **目標**：請在HW1繳交的文件基礎(CVPR格式)之上，加入到10/30為止的Project進度，***新增或修改的地方請特別以藍色字體表示***，我們會比對HW1與HW3的進度差異進行評分，有越詳盡的Project內容及更多的進度為優

1. HW3 內容必須包括
    1. Title : project的題目。
    2. Team member : 隊員名稱，包括學校、學號、以及姓名 (中文 如 : 國立成功大學A12345678 王小明)。
    3. Abstract (10%) : 簡要描述研究主題、預期的技術、預計的結果
    4. Introduction (20%) : 需詳細描述研究動機、問題定義、現有的解決方法跟文獻，以及你預期的解決方法
    5. Method  (50%) : 需要盡可能詳細描述你的技術細節，也可繪製架構圖、並請確保技術的可行性及創新度。
    6. Experiment  (15%) : 需要詳細描述你的實驗方法及設置，像是使用的資料集，也需要說明你的實驗細節，如：執行環境，若有實驗結果也可寫入。
    7. Reference (5%) : 參考文獻
