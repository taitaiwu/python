# Homework1: 期末專案計畫

> **目標**：以組為單位，規劃並繳交Final project內容，以CVPR格式寫成一到兩頁的提案報告(以英文撰寫為佳)。

1. 選擇自行提出主題的隊伍，每組人數最多四人，請明確註明每人的中文姓名、學校名稱、以及學號。
2. 內容必須包括
    1. Title : project的題目。
    2. Team member : 隊員名稱，包括學校、學號、以及姓名 (中文 如 : 國立成功大學A12345678 王小明)。
    3. Abstract  (10%) : 簡要描述研究主題、預期的技術、預計的結果
    4. Introduction (45%) : 需詳細描述研究動機、問題定義、現有的解決方法跟文獻，以及你預期的解決方法
    5. Method (30%) : 需描述預期開發的技術、盡可能描述該技術細節。
    6. Expected Results (10%)：需描述預期可得到的結果，可先畫示意圖表示、或手繪想像的結果、或從網路上找到預期的效果等等，描述越具體越好。
    7. Reference (5%) : 參考文獻
