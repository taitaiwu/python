# 臺灣現存大專院校未來退場機率預測
國立彰化師範大學 資訊工程學系 吳佳泰
國立彰化師範大學 資訊工程學系 陳佳君

[YouTube介紹](https://www.youtube.com/watch?v=3aTvzaLG1mE)

## 摘要
近年來，臺灣因少子化以致學生數下降，根據新聞報導，2028年教團預估有40所大專院校恐面臨退場危機，因此我們想要透過一些開放平台資料來預測全台140間大專院校(扣除兩間空中大學及九間宗教研修學院)未來退場的機率。

## 介紹
少子化已成為現在的社會議題，大專院校招生名額數供過於求，教育部預測114學年度大一新生人數將跌破20萬人接下來，117學年逢虎年效應，人數將減至17.3萬人低點 [1]，對於大專院校而言，這是對於他們永續發展的嚴峻挑戰，未來將會有越來越多學校面臨轉型、整併或是退場的危機，甚至有教團評估未來恐有40所大學面臨退場。身為學生，學校與我們息息相關，因此我們想要透過本次研究預測臺灣現存大專院校未來退場機率為何。

![大學新生推估人數](大學新生推估人數.png)

在此背景下，我們想要運用各校現有且公開的資料(如:新生註冊率、學生退學率、就學穩定率、學校可用資金、入學分數等)、近年退場學校(如:明道大學、環球科技大學等)的相關資料，以及未來即將停辦之學校(如:華夏科技大學、中華大學)，來作為我們的輸入，輸出是預測每間學校退場機率，希望藉由建立一個模型來預測學校的退場風險，並找出最重要的影響因素，協助理解哪些條件與退場最有關聯。

目前，教育部會依據學校財務狀況等指標，採取規則式的方式監管大專院校，但並沒有發展出想關的AI模型預測系統來做預測。因此，我們希望嘗試利用機器學習方法，整合多維度的特徵，來建構一個更全面的退場風險預測模型。

現今不僅只是台灣，日本、韓國、美國等國家也因少子化的問題，而導致私立大專院校經營困難。日本擁有專責機構，並發展經營判斷指標來評估；韓國有絕對評鑑制度；美國則是將校務資訊公開化 [4]。

在陳昀陞的研究中[3]，使用遞迴模型與類神經網路來預測大專院校未來註冊率，並依當時法規明確列出面臨危機的學校；在程冠瑜的研究中 [4]，使用多種統計與機器學習方法（逐步迴歸、PCA、決策樹、隨機森林、SVM 等），並處理了資料不平衡的問題，但結論並未說明有哪些私校被預測可能退場。

我們預期利用電腦演算法透過分析歷史資料來預測未來事件或行為。第一步先蒐集、整理資料；第二步做建立可用於建模的乾淨資料集；第三步模型建置與調優，完成隨機森林模型訓練，最後分析每間學校的退場風險之高低程度。

## 方法
在程冠瑜的研究中提到使用隨機森林的方式成效最佳 [4]，因此我們也想嘗試利用隨機森林的方式來做訓練，觀察我們訓練後的成果。

我們的研究方法及步驟大致如下方流程圖，先蒐集資料，再進行整理及特徵建立，接著開始模型訓練，評估此模型，最後解釋並分析預測結果。

![流程圖](flow_chart.png)

我們繪製了下圖的第一棵決策樹，並依決策樹的內容來蒐集資料，並將資料存放於 [university_data](https://docs.google.com/spreadsheets/d/1LTh8cjyMFGgFueylPxTk9zG9fGpqFgid60u8pbg9XPY/edit?usp=sharing) 內，我們一共蒐集了 16 項資料，每個 column 的資料種類具體如下表所示。

![決策樹](決策樹.png)

| 欄位名稱 | 類別 | 說明 |
|---|---|---|
| id | int | 資料編號 |
| data_year | int | 資料年份 |
| school_name | string | 學校名稱 |
| private_flag | int | 私立大學指標（0：公立；1：私立） |
| school_type | string | 學校種類（一般大學／技專校院） |

接著在 Google Colab 中進行後續程式步驟 (詳情見 [Predicting the Probability of University Closure in Taiwan](https://colab.research.google.com/drive/1na5kfOI5ynVIbYxAQek6QnjMQMlWKDp6?usp=sharing))，我們在此檔案中做資料預處理，包括缺失值處理、標準化及拆分資料集，由於資料可能會存在不平衡的問題，退場學校和現存學校數量差距過大，因此我們需要將資料做平衡處理，以免隨機森林可能會傾向「全部預測為續存」。資料將調整類別權重，少數類別（退場）要更重要；多數類別（續存）相對權重要降低，以降低類別不平衡對模型造成的偏誤，並提升對退場類別的辨識能力。

隨機森林透過多棵決策樹的投票機制進行分類，能有效降低單一決策樹的過度擬合問題。隨機森林亦可提供特徵重要性排序，有助於解釋哪些因素對大專院校退場風險影響較大。

最後解讀模型，利用精確率、準確率等對模型進行評估，將先不將最近一年的資料投入模型訓練，使用先前資料訓練後，再與最新的資料做比對。

## 實驗
我們蒐集了每間學校在 106 ~ 112 年間的資訊(university_data.csv)。其中學生類及財務類相關資訊大部分由大專校院校務資訊公開平台[5]取得，少部分學校缺失值我們去搜尋了各校官網的財政資料，包含了以下資料
* 臺北市立大學 : 因為該校並非國立亦非私立大學，在大專校院校務資訊公開平台無法取得此校的財政類資料，所以我們使用該校校網的歷年財政資料做為缺失值資料[8]。
* 明道大學 : 因不明原因，在該校尚未發出停招或停辦的消息時，大專校院校務資訊公開平台無法取得該校 111 年財政類資料，所以我們使用該校校網 111 年的會計月報做為缺失值資料[9]。

學校所位地區分級我們採用侯佩君的研究資料[7]，由於每間學校可能有許多不同的校區，因此我們選擇每間學校的校本部或最主要的校區做為依據。此外，我們針對以下學校做了缺失值處理
* 國立金門大學 : 位於金門縣金寧鄉，並未被列入侯佩君研究的分級裡[7]，因此我們將金寧鄉與澎湖縣的馬公市(第 4 級)、西嶼鄉(第 6 級)比較，最終將其列在第 5 級

專案輔導學校由於教育部並不公開歷史資料，因此我們透過前幾年的報導做為我們的資料依據[11][12][13]，但我們取得的資料有限，所以只有報導中明確指出的專輔學校與年份，該筆資料的專案輔導學校指標才會被列為1。
	
此外，由於高苑科技大學(現為台鋼科技大學)在 2022 年時被認定惡意積欠薪資、爆發血汗學工案等負面新聞，因而被列入專輔學校，同年底第11屆學校董事會全體董事均已辭職，瀕臨退場狀態[14]，若非在即將被勒令停招前夕獲得台灣鋼鐵集團資助新台幣 2.8 億元[15]，該校早已退場，這並非常理，因此我們將台鋼科技大學 111 年之 closure_flag 設為 1，但其他年度的 closure_flag 依舊設為 0。

程式語言我們選用 Python 來實作，在 Scikit-learn 函式內的隨機森林模型，讓我們可以使用 sklearn.ensemble 裡的 RandomForestClassifier（分類問題）或 RandomForestRegressor（回歸問題）。選用Scikit-learn 的原因是其上手容易、語法清晰、內建功能完整以及模型評估工具齊全，適合我們這種初學者以及小專案來做使用。

針對訓練資料，我們做了預處理，包括去除無用資料 (id、data_year 與 school_name) 和對學校總類進行編碼。之後我們執行了第一次程式，輸入了 1043 筆資料(包含17 筆退場學校與 1026 筆續存學校)，針對我們的資料取得下表的數據，我們可以從中得知，測試集裡完全沒有退場學校，即使我們已經做了平衡處理。所以 accuracy = 0.98 是假象，因此我們可能需要針對訓練資料再做處理。

我們一共反覆測試了 10 次，accuracy 的區間落在 0.85 左右，在訓練資料極度失衡的情況下，已經是不錯的結果了。大部分 closure_flag=1 時，recall = 0.67 表示我們有特別突出少數種類，因此這個模型可以作為評判現存大專院校退場機率的系統。此外，我們也做了特徵重要性分析，發現 new_student_count、enrollment_quota 與 enrollment_rate 是模型中重要性最高的三個特徵，而 private_flag、school_type 與 totur_flag 則是重要性最低的三個特徵。

## 預期結果
我們預測經平衡處理後的隨機森林模型的準確率可能會下降，但召回率可能會顯著上升，模型會更注重於退場學校，我們也可以利用隨機森林的特徵重要性中得知影響退場的關鍵因素，最後我們會得到每間學校的退場機率，並且可以根據機率來排名哪些學校處於高風險，輸出的結果為 0 ~ 1之間的小數，我們可能將數值轉換成百分比的形式，或是以數值區間來標示低度風險、中度風險及高度風險。

模型預測的結果是符合預期的，我們不需要追求 100% 準確率的模型，重點是模型能有效識別高風險學校，具有退場風險的大多是技專校院，尤其是規模較小、科系較單一的學校，通常公立學校可能最後會和他校整併或合併，而私立學校最終則可能會面臨退場。

未來我們可以將 urbanization_level 使用各校區人數比例來算出加權後的平均等級，讓此欄位變得更精確；更嚴謹的蒐集資料，以確保訓練模型的可靠度；蒐集更多樣化的資料，讓模型可以從更多面向分析，考慮到更多情況；針對每個校系做分析，讓預測變得更加精確；搜尋最新的財務資料來替換，讓測試資料更符合現況。
